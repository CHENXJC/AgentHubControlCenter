## Short Project Summary

AgentHubControlCenter is a Streamlit-based AI AgentOps dashboard that organizes
11 local AI Agent and Skill projects into one portfolio command center. It
shows manifests, safe actions, useful signals, connector readiness, local
workflow simulations, approval gates, Codex prompt handoff, report export, and
public showcase readiness without executing real actions or connecting live
accounts.
